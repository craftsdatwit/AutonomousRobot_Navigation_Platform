HOW TO INSTALL THESE FILES
 After following the root readme.

1.) The warehouse.world file must go in the ~/catkin_ws/src/turtlebot3_simulations/turtlebot3_gazebo/worlds folder

2.) The warehouselaunch.launch file must go in the ~/catkin_ws/src/turtlebot3_simulations/turtlebot3_gazebo/launch folder

3.) Go to ~/catkin_ws and run source devel/setup.bash

4.) To run, enter roslaunch turtlebot3_gazebo warehouselaunch.launch
