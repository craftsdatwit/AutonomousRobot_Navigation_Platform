HOW TO INSTALL THESE FILES
 After following the root readme.

1.) The demoworld.world file must go in the ~/catkin_ws/src/turtlebot3_simulations/turtlebot3_gazebo/worlds folder

2.) The warehousemultirobot.launch file must go in the ~/catkin_ws/src/turtlebot3_simulations/turtlebot3_gazebo/launch folder

3.) Please be advised that the launch file contains a path to a file called "robots5.launch" please modify this with the name of the launch file you created during the multi-robot setup.

3.) Go to ~/catkin_ws and run source devel/setup.bash
